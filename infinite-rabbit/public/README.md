# 🐰 동물 친구들 무한의 계단 (Animal Friends Infinite Stairs)

![Game Banner](https://img.shields.io/badge/Game-Infinite_Stairs-orange?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-HTML5_Canvas_|_JS-blue?style=for-the-badge)

귀여운 동물 친구들과 함께 끝없이 이어지는 계단을 올라보세요! 
방향을 잘 맞추어 점프하고, 에너지가 바닥나기 전에 아이템을 획득하며 최고 기록에 도전하는 중독성 강한 캐주얼 게임입니다.

## 🎮 주요 게임 특징

- **3종의 개성 넘치는 캐릭터**: 분홍토끼, 아기팬더, 아기백호 중 원하는 친구를 선택할 수 있습니다.
- **다양한 테마 배경**: 캐릭터를 선택하면 그에 맞는 배경(하늘, 대나무 숲, 호수)과 계단, 아이템이 자동으로 변경됩니다.
- **역동적인 애니메이션**: 
  - **팬더**: 점프 시 몸이 늘어나고 기울어지는 쫀득한 물리 애니메이션!
  - **호랑이**: 스프라이트 시트를 활용한 박진감 넘치는 도약 애니메이션!
- **생동감 넘치는 연출**: 움직이는 구름, 날아다니는 새, 물결치는 호수, 흔들리는 대나무 숲 등 디테일한 배경 효과를 감상하세요.
- **점진적 난이도**: 멀리 갈수록 시간이 더 빠르게 줄어듭니다. 당신의 순발력을 시험해보세요!

## ⌨️ 조작 방법

- **PC**: 
  - `⬅️` 또는 `A`: 왼쪽 점프
  - `➡️` 또는 `D`: 오른쪽 점프
- **Mobile**:
  - 하단의 `왼쪽`, `오른쪽` 터치 버튼 사용

## 🚀 실행 방법 (중요)

본 게임은 브라우저에서 실행됩니다. 하지만 게임 내 **에셋 투명화 처리(CORS)** 기능을 온전히 사용하기 위해서는 로컬 서버 환경에서의 실행을 권장합니다.

### 방법 1: HTTP 서버로 실행 (권장)
CORS 보안 정책 이슈 없이 모든 그래픽 효과를 완벽하게 즐길 수 있습니다.
```bash
# Python이 설치된 경우
python -m http.server 8080

# Node.js(http-server)가 설치된 경우
npx http-server ./
```
이후 브라우저에서 `http://127.0.0.1:8080` 으로 접속하세요.

### 방법 2: 로컬 파일 직접 실행
`index.html` 파일을 더블 클릭하여 바로 실행할 수 있습니다.
- 단, 브라우저 보안 정책에 따라 캐릭터 배경이 투명하게 보이지 않을 수 있으며, 이 경우 안내 팝업이 노출됩니다.

## 📂 프로젝트 구조

- `index.html`: 게임의 메인 레이아웃 및 UI 구조.
- `game.js`: 렌더링, 애니메이션, 충돌 판정 등 핵심 로직.
- `style.css`: 게임 디자인 및 레이아웃 스타일.
- `assets/`: 캐릭터 및 배경 이미지 리소스.
- `assetsData.js`: 에셋의 Base64 데이터 (로딩 최적화용).
- `PRD.md`: 상세 제품 요구 사항 문서.
